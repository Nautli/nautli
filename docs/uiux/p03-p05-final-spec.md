## 구현 스펙

### A. recall 이벤트 로그 (코어, 전제 작업)
- recall 실행 시(원천 무관: MCP/CLI) `~/.nautli/events/`에 recall 이벤트 1줄 기록: `{type:"recall", query, scope, hits:[fact_id...], source:"mcp"|"cli", at}`.
- 불변식 유지: 이벤트는 append-only, 기억(fact) 자체는 무변경. 프라이버시: 로컬 파일뿐.
- 대시보드 `/api/activity?since=` 로 최근 remember/recall 이벤트 폴링 가능.

### B. P0-3 연속성 스텝 "첫 진짜 기억" (Setup 필수 4개 완료 후 히어로)
1. **와우 순서 원칙(페르소나 판정)**: 1차 와우 = 검진 리포트의 "내 볼트에서 나온 진짜 모순"(이미 구현, 수동적·1분 내). 이 스텝은 2차 와우(능동 검증)로, 검진을 했든 안 했든 Setup 완료 후 노출.
2. 스텝 A (앱 전환 1회, 마지막):
   - 카피: "Claude Code에 이 한 줄을 붙여넣으세요" + [복사] — 문구는 유효 기억 예시 3개 중 택1 표시(예: "기억해줘: 나는 커밋 메시지를 한국어로 쓴다" / "기억해줘: 내 기본 브랜치는 main이다" / 직접 입력). 붙여넣는 대상이 실제 습관이 되도록.
   - 대기 UX(전문가 지적): 폴링 중 단계 카피 전환 — 0~15s "붙여넣으셨나요?" → 15~60s "Claude가 응답하면 자동으로 감지됩니다" → 60s+ "감지가 안 되면 MCP 연결 문제일 수 있어요 [진단 실행]". 스피너만 금지.
   - 감지: /api/activity 폴링으로 source=mcp remember 이벤트 감지 → "저장 감지 ✓ Claude가 nautli에 썼어요".
3. 스텝 B (앱 전환 0회): "이제 nautli가 직접 읽어볼게요" [읽어보기] → 대시보드가 recall API 실행, 방금 fact를 실물 표시 + 카피: "이 기억은 Cursor든 새 세션이든 어디서 물어도 똑같이 나옵니다."
4. 스텝 C (선택 심화, Cursor 연결 ✓인 유저에게만): "회의적이라면: Cursor에서 직접 물어보세요" + 질문 [복사] → recall 이벤트(source=mcp, 다른 세션) 감지 시 "방금 두 도구가 같은 뇌를 썼습니다" 배너.
5. 폴백: 각 스텝에 [건너뛰기] — 완료 아님을 정직 표기(체크 아이콘 대신 "건너뜀" 회색). 거짓 완료 금지.
6. 기존 "체험 소화"(합성 샘플)는 선택 섹션으로 강등 유지.

### C. P0-4 공유 카드 (검진 리포트에서)
1. 헤드라인 = 발견 프레임: "My AI memory had **N contradictions** and **M duplicates** I didn't know about. Found in X min." 점수는 하단 소형 배지(score/100 + "sampled 40 notes" 라벨 병기).
2. [Share card] 버튼 → 미리보기 모달 → PNG 다운로드. 렌더는 로컬 canvas, 서버·외부요청 0.
3. 포함 필드 고정: 발견 카운트 3종(중복/모순/junk%) + 표본 라벨 + 소요시간 + nautli 로고 텍스트 + CTA "What's hiding in yours? npx nautli dashboard". claim·경로·프로젝트명·원문은 **필드 자체가 없음**(옵션 미제공).
4. 옆에 X/Threads용 텍스트 스니펫 [복사]: 카드와 동일 프레임 1~2문장 + 커맨드.
5. 언어: 대시보드 로케일 따름, CTA 커맨드는 항상 영문. before/after 카드는 v0.2 자리만(재검진 기능 의존) — UI에 미노출.
6. ⚠️ 렌더 산출물 → 빌드 후 비주얼 QA 게이트(상태: 카운트 0건/큰 수/한글/영문 4매트릭스 스크린샷 검안) 필수.

### D. P0-5 검진 하드닝
1. 시작 모달 사전검사: python3 + **claude CLI 로그인**(doctor 로직 재사용). 실패 시 시작 비활성 + 사유 + **복구 버튼**([claude 설치 안내] / [로그인 명령 복사]). 
2. 데이터 경계 카피 명확화(페르소나 1순위 요구): "선택한 폴더의 노트 **텍스트가** 내 Claude 구독을 거쳐 Anthropic에서 처리됩니다. 요약·점수만 로컬에 저장되고 어디에도 업로드되지 않습니다." 모호한 "판정 텍스트만" 표현 금지.
3. 스캔 대상 사전 표시: 감지된 최상위 폴더 **체크박스 목록**(기본 전체 선택, 해제=제외) + 예상 파일 수 + **볼트 크기 기반 예상 시간**(실측 30노트≈8분 외삽).
4. 표본: 경로순 → 시드 고정 셔플(시드=볼트 경로 해시, 재현성). 리포트·공유카드에 표본 크기 상시 라벨.
5. 부분 결과 스트리밍: 진행 바 아래 "지금까지: 모순 N건 · 중복 M건" 실시간 갱신(10노트 단위). 첫 발견 1건은 티저로 실물 미리보기.
